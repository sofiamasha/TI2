# 🚀💖 Trabalho Interdisciplinar II (TI II) — Back-end

💻 Repositório dedicado ao desenvolvimento do projeto **SafeHer**, construído ao longo da disciplina de Trabalho Interdisciplinar II — PUC Minas

---

## 🌸 Sobre a disciplina

O TI II tem como objetivo integrar conhecimentos adquiridos ao longo do curso, com foco no desenvolvimento **back-end** de aplicações.

A proposta é evoluir um projeto já existente, aprofundando aspectos técnicos e estruturais.

---

## 🧠 Sobre o projeto — SafeHer

O **SafeHer** é uma plataforma digital criada com o propósito de oferecer um ambiente seguro e acolhedor para relatos de situações sensíveis no contexto profissional.

O sistema permite:

* Compartilhamento de experiências de forma segura
* Organização e armazenamento de dados
* Estruturação de informações para futuras análises

💡 Este projeto é uma evolução do trabalho desenvolvido no semestre anterior, agora com melhorias na parte lógica e estrutural.

---

## 👥 Equipe

Projeto desenvolvido em grupo (4 integrantes), com colaboração na construção das funcionalidades e organização do sistema.
